🏫 Import skupinek (vytvoření/aktualizace)

Povinné sloupce:

  • name - Název skupinky
  • ageGroup - Věková skupina (např. 1-2 years)
  • color - Barva v hex formátu (např. #FF6B6B)

Volitelné sloupce:

  • description - Popis skupinky
  • dayOfWeek, time, location, capacity, startDate, endDate - Pro automatické vytvoření lekcí

Existující skupinka se stejným názvem bude aktualizována.
